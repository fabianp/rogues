rogues.matrices.qmult¶

rogues.matrices.qmult(b)¶
QMULT Pre-multiply by random orthogonal matrix.

QMULT(A) is Q*A where Q is a random real orthogonal matrix from the Haar distribution, of dimension the number of rows in A. Special case: if A is a scalar then QMULT(A) is the same as

QMULT(EYE(A)).

Called by RANDSVD.

Reference: G.W. Stewart, The efficient generation of random orthogonal matrices with an application to condition estimators, SIAM J. Numer. Anal., 17 (1980), 403-409.
